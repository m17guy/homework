Events angular -
click-
<button (click)="title='Hello'">Change title</button>

באופן דומה ניתן לקרוא לפונקציה במקום-
<button (click)="changeTitle()">Change title</title>

ואז נכתוב את הפונקציה בקובץ app.component.ts בתוך הקלאס שלנו -
changeTitle() {
this.title = "Hello";
}

באנגולר, אם נרצה שיתבצע אירוע מסוים, נשים את האירוע בתוך סוגריים, ללא המילה on.
כלומר onClick יהפוך לclick וכו'..
אין צורך במילה function לפני הפונקציה, זו כתיב השל ES6.
