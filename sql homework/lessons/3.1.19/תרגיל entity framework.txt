1. צור בסיס נתונים קטן שיקרא בשם עטים. לעט יש שם, ID, מחיר. הוסיפו לטבלה 3 עטים.
2. הוסיפו מודל entity framework לפרוייקט.
3. כיתבו קוד שידפיס את שמות העטים על המסך.
4. הוסיפו קוד שישנה את מחיר אחד העטים (לפי בחירתכם) לכפול.
5. מחקו עט אחר.
6. הוסיפו עט נוסף.
7. דאגו שכל השינויים יסונכרנו מול בסיס הנתונים.
8. כעת בדקו ב-sql object explorer שהשינויים אכן בוצעו.
